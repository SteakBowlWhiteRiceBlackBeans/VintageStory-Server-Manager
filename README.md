# Vintage Story Server Manager

A lightweight Windows application for managing a **Vintage Story dedicated server**.

It provides a simple graphical interface to:
- Start, stop, and restart the server
- Configurable server data location with field for launch arguments
- View live server console output
- Send server commands
- Manage auto saves, auto backups, and a scheduled restart
- View installed server-side mods
- View current player count
- View CPU and RAM usage
